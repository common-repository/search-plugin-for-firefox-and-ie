== Description ==

This plugin allows your visitors to easily install the search bar of your site in their browser. The plugin also allows you to customize the search bar with the name and image on your web site.

== Installation ==

This section describes how to install the plugin and get it working.e.g.1. 

Unzip the file and upload entire `search_for_browser` folder to the `/wp-content/plugins/` directory2. 
Activate the plugin through the 'Plugins' menu in WordPress3. 
Go to the Settings > Search Browser and configure the plugin options.

== Widget installation ==  

Activate the plugin, then simply go the Presentation > Widgets section and drag over the Search Button for browser Widget.You can easily customize the widget by entering the URL of an image that you want to show to your visitors for install the search bar in their browser.

== Manual installation ==  

Activate the plugin, then go to the Settings > Search Browser and configure the plugin options. After press on "Update Option" button you can find the php code to integrate the plugin in your theme.
